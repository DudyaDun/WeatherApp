# WeatherApp — Приложение для отображения метеоданных

Приложение WinForms (.NET 6, C#) для хранения, просмотра и редактирования
почасовых метеорологических наблюдений, совместимое с форматом экспорта Open Meteo.

---

## Требования

| Инструмент              | Версия   |
|-------------------------|----------|
| .NET SDK                | 10.0+    |
| Windows                 | 10/11    |
| Visual Studio 2022      | или Rider / VS Code |

---

## Быстрый старт

### 1. Восстановить пакеты и запустить

```cmd
cd WeatherApp
dotnet restore
dotnet run
```

### 2. Или открыть в Visual Studio

Откройте `WeatherApp.slnx`, нажмите F5.

## Функциональность

### Просмотр данных
- Таблица с почасовыми записями (дата, время, температура, влажность, облачность)
- Цветовое кодирование температуры (синий → зелёный → жёлтый → красный)
- Вычисляемые поля: «Состояние неба» и «Индекс комфорта»
- Фильтрация по диапазону дат

### Ввод и хранение данных
- Импорт Excel-файлов в формате Open Meteo (*.xlsx)
- Данные хранятся в JSON: `%AppData%\WeatherApp\weather_data.json`
- Дублирующиеся записи при повторном импорте не добавляются
- Экспорт видимых записей в CSV (Excel/LibreOffice совместимый)

### Редактирование объектов
| Действие               | Способ                         |
|------------------------|--------------------------------|
| Добавить запись        | Кнопка ➕ / меню / Insert      |
| Изменить запись        | Двойной клик / F2 / кнопка ✏ |
| Удалить записи         | Кнопка 🗑 / Delete             |
| Изменить локацию       | Кнопка «⚙ Изменить локацию»   |
| Добавить локацию       | Меню «Данные → Новая локация» |
| Удалить локацию        | Меню «Данные → Удалить локацию» |

### Структура данных

**WeatherLocation** — метаданные точки наблюдения:
- Название города/станции
- Широта и долгота (WGS-84)
- Высота над уровнем моря (м)
- Разрешение данных (hourly / daily / …)

**WeatherRecord** — одна запись наблюдения:
- Временна́я метка
- Температура воздуха на высоте 2 м (°C)
- Относительная влажность на высоте 2 м (%)
- Общая облачность у поверхности (%)
